About OCTest

OCTest 是一个轻量但可持续演进的静态站点项目，目标是验证“想法 → 开发 → 发布”的快速交付闭环。 当前包含欢迎页、博客模块、多页面信息架构，以及版本变更记录。

这个仓库强调三件事：快速试验、可读结构、持续迭代。

Tech Stack

  • Static HTML / CSS / JavaScript
  • GitHub Repository + Git versioning
  • GitHub Pages deployment
